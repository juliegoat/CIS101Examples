There are two types of lists in HTML

This is an ordered list. It uses numbers to specify the order of the items in the list. OL stands for Ordered List.

    This is the title of the list, if it has one.
  1. This is the first list item
  2. This is the second list item
  3. This is the third list item
  4. I think that you get the picture...

The other type of list is an unordered list. This list has a bullet for each item, indicating that there is no numerical order to the items. UL stands for Unordered List.

    This is the title of the list, if it has one
  • This is a list item
  • This is also a list item
  • This is yet another list item
  • Again with the list items
